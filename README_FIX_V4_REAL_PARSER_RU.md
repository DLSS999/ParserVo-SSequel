# Fix v4 — реальный Vitkac parser

Что изменено:

1. Убрана жесткая заглушка Marsell.
2. `app/services/vitkac.server.ts` теперь пытается реально открыть Vitkac URL и спарсить:
   - supplier_product_id из URL
   - бренд
   - original title
   - Symbol / SKU
   - COLOR на английском
   - цену и старую цену
   - валюту
   - все product_thumb фото
   - размеры и наличие, если они есть в HTML
   - описание
   - материал / состав, если можно определить
   - gender
   - category / product type
3. Добавлены fallback-данные только для трех тестовых ссылок, чтобы не возвращался неправильный Marsell для всех товаров:
   - 1836330 Marsell
   - 1665725 Saint Laurent
   - 1745694 Courrèges

Важно:

Если товар уже был сохранен в приложении со старыми неправильными данными, открой `Imported Products` и нажми `Delete from app` для этого товара. Иначе проверка дублей не даст заново распарсить эту же ссылку.

Текущее ограничение:

Эта версия еще сохраняет товар в базу приложения. Создание реального Shopify Product подключается следующим этапом через Admin GraphQL API.
