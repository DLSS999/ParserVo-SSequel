# Stone Island Shopify import fix

Исправлена блокировка импорта товара с ошибкой:

`Нет проверенных фотографий текущего товара.`

## Что изменено

- Shopify strict sync теперь принимает официальные изображения Stone Island с `stoneisland-cdn.thron.com`.
- Удалена подмена URL через фиктивный query-параметр `parservo_media`.
- В модель источников добавлен `STONE_ISLAND`.
- Оригинальные публичные HTTPS-фотографии передаются Shopify напрямую.

Проверочный HTML товара содержит Product JSON-LD с 7 официальными изображениями THRON, ценой 924 PLN и описанием товара. Парсер ограничивает импорт первыми 5 изображениями согласно текущей логике приложения.
