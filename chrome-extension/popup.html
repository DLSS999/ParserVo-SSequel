ParserVo Vitkac Capture

Для массового импорта: загрузи Excel в Shopify app, затем нажми Start automatic import. Для контроля наличия нажми Start stock refresh — расширение откроет уже импортированные товары Vitkac, обновит наличие и при включенной синхронизации отправит остатки в Shopify.

API settings

Бери из PowerShell после запуска shopify app dev. Можно вставить базовый URL или полный URL с /api/vitkac-capture.
