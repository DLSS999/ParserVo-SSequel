# ParserVo v43 — Imported Products Pagination

Фикс ускоряет страницу **Imported Products** для больших каталогов.

## Что изменено

- На странице больше не загружается весь список 2000+ товаров сразу.
- По умолчанию загружается только **50 товаров на страницу**.
- Добавлены переключатели: **50 / 100 / 200 rows**.
- Добавлена навигация: **First / Prev / Next / Last**.
- Фильтры и массовое выделение работают по текущей загруженной странице, чтобы интерфейс не подвисал.
- При переходе между страницами выделение сбрасывается, чтобы случайно не удалить товары с другой страницы.
- В ZIP также сохранен фикс v42 для Shopify `UserError code` в синхронизации остатков.

## Как установить

Распаковать ZIP в корень проекта с заменой файлов:

```powershell
cd "C:\Users\user\ParserVo CINQ\supplier-import-sync"

npm run build
git add -A
git commit -m "Add pagination to imported products page"
git push origin main
```

После деплоя Vercel открыть приложение и нажать **Ctrl + F5**.

## Важно

Если нужно искать товар по всему каталогу, лучше использовать поле Search на странице после перехода на нужную страницу или добавить серверный поиск отдельным следующим фиксом. Этот фикс в первую очередь решает долгую загрузку за счет ограничения списка до 50 товаров на страницу.
