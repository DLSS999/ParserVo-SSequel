# ParserVo v48 — Warehouse Transfer Query Cost Fix

Исправляет ошибку Shopify GraphQL:

```text
Query cost is 1370, which exceeds the single query max cost limit (1000)
```

## Что изменено

1. Warehouse Transfer больше не запрашивает `inventoryLevels(first: 50)` для каждого варианта.
2. Теперь запрашиваются только 2 нужных склада:
   - склад-отправитель
   - склад-получатель
3. Максимальный batch для переноса склада ограничен до 5 товаров за запрос.
4. В интерфейсе Warehouse Transfer оставлены безопасные размеры партии:
   - 1 товар
   - 3 товара
   - 5 товаров

## Как пользоваться

1. Открой `ParserVo → Warehouse Transfer`.
2. Выбери склад, с которого переносим.
3. Выбери склад, на который переносим.
4. Выбери тег Shopify.
5. Поставь `Batch size: 3` или `5`.
6. Сначала нажми `Preview first batch`.
7. Если товары правильные — нажми `Start full transfer by tag`.

## Важно

Для больших тегов нельзя запрашивать слишком большую партию, потому что Shopify считает стоимость GraphQL-запроса по вложенным товарам, вариантам и остаткам. Поэтому перенос идет маленькими партиями, но автоматически до конца.
